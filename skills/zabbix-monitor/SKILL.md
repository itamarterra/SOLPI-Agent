# Zabbix Monitor (Local)

Habilidade para transformar o computador local em um "host" monitorado pelo SOLPI.

## Funções
- Ler uso de CPU/RAM.
- Detectar falhas de processo.
- Reportar para o endpoint `/zabbix/ingest` do plugin.

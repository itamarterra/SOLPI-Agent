# Auto-Evolution

Este é o código de conduta do Agente quando está em Modo Livre.

## Diretrizes Autônomas
1. **Silêncio Operacional:** Trabalhe em segundo plano sem interromper o Itamar, a menos que uma "Ouro" (Oportunidade) seja encontrada.
2. **Ciclo de Feedback:** Teste cada novo comando antes de integrá-lo à sua base de conhecimento definitiva.
3. **Escalabilidade:** Se uma tarefa for muito grande, crie um script Python temporário para executá-la em paralelo.

## Objetivos de Fundo
- Monitorar preços de fornecedores.
- Buscar erros nos logs do GLPI/Zabbix.
- Gerar ideias de posts virais baseados no Google Trends.

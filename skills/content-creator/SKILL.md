# Content Creator

Habilidade de produção e consumo de mídia do agente.

## Capacidades
1. **Trend Hunting:** Abrir canais de tendências e ler comentários/títulos para identificar o que está viral.
2. **Video Editing:** Usar FFmpeg para compor clips curtos (TikTok/Reels style).
3. **Media Control:** Pausar, avançar e ajustar som de vídeos e músicas.

## Workflow de Viralização
1. Pesquisar trends no YouTube/Twitter.
2. Gerar roteiro persuasivo com IA.
3. Montar vídeo usando `create_viral_video`.
4. Sugerir hashtags e títulos para postagem.
